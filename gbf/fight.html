Turn Flow
Preparation Phase
(Manual)		 Use Skills and Summons
Attack to proceed to the Attack Phase
Attack Phase
(Automatic)		 For each Ally :
  • If 100% Charge Bar and Charge Attack is on Auto → Use Charge Attack
  • Else → Use Normal Attack
  • If Allies used two or more Charge Attacks → Use Chain Burst
For each Foe :
  • If Mode Bar has been filled → Enter Overdrive Mode
  • Else if Mode Bar has been drained → Enter Break Mode
  • If Phase Change has been triggered → Change Phase
For each Foe :
  • If Special Attack has been triggered → Use Special Attack
  • Else if Charge Diamonds are filled and Foe is not in Break Mode → Use Special Attack
  • Else → Use Normal Attack
